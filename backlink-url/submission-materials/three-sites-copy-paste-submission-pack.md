## Universal Safety Notes

- If a directory requires login or OAuth, the login must be completed by a human.
- If a directory shows CAPTCHA, Turnstile, Cloudflare challenge, or bot detection, stop and handle manually.
- If a directory asks for paid placement, sponsored listing, payment card, or checkout, do not continue unless payment is explicitly approved.
- If a directory asks for a reciprocal backlink, mark it as manual-review or skip unless reciprocal linking is intentionally approved.
- If the form has a final submit button, review every field before clicking.
- Use clean URLs only. Do not add `ref`, `utm`, or tracking parameters unless intentionally required.
